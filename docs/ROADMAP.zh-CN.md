# 路线图

**语言：** [English](ROADMAP.md) | 中文

本路线图描述 `pricereader` 公开仓库的当前状态，并区分已经可用的内容和每个使用者需要在自己环境里补完的内容。

## 已经足够使用

- 规则引擎和适配器
- 运行时配置 API
- 调度和通知流程
- pytest 测试套件

## 需要本地补全

- 私有来源/规则配置
- 生产通知目标
- 针对你的来源列表设置限速和礼貌抓取策略
- 如果暴露到本机以外，需要增加鉴权

## 建议改进

- 增加新的来源适配器
- 增加运行时配置编辑的 UI 测试
- 改进重复项/静音流程
- 为共享部署增加鉴权和审计日志

## 仍值得补充的文档

- 使用非私有演示数据制作截图或短录屏。
- 为常见请求/响应补完整 API 示例页。
- 为持久化数据路径补备份和恢复说明。
- 等公开用户开始部署后，基于真实问题整理排障页。

## 维护说明

- 公开示例保持通用。
- 英文和中文说明保持同步。
- 尽量拆小 issue 和 pull request，方便 AI 辅助贡献者安全接手。
- 发布新版本前重新跑敏感信息扫描。
